Command-Line Utilities

  • regesc - This is a simple CLI program I made to sanatize a string so it can be used in a regex as just a string. It just takes a line from stdin and prints out an escaped version. It's so simple that the C file is smaller than the README.
  • cowfriends - This is a very simple program I made because I wanted to see all of the different options for cowsay without typing each one out manually. It's a single file C++ program. It should work on macOS (homebrew or macports) and probably Linux, so long as your package manager has put cowsay's cowpath in /usr/local/opt/cowsay/share/cows or /usr/share/cowsay/cows
